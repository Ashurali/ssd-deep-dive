# The Vt Distribution Playground

The diagram of the entire flash half of the book, made draggable: program a single cell, watch 4,000 cells pile into a bell curve, cram 2/4/8 bells into the same window (SLC/MLC/TLC), then drag P/E cycles, retention and read count to drift the bells into each other while a live RBER counter ticks. Read Retry re-centers the references — until the bells merge, which is the whole limit of hard-decision reading.

**Book anchor:** Realizes Figs 3-1→3-5, 3-44→3-48, 3-50→3-57, 3-60 and 1-24→1-27 (CH3 pp. 1–13, 44–63) — atlas Cluster A.
