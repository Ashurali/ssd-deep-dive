# How ECC finds and fixes a bit — Hamming visualizer

An interactive Hamming(12,8) visualizer: flip any data bit and watch the parity checks light up to *announce the flipped bit's own location*, then correct it. The clearest way to internalize why a syndrome pinpoints a single-bit error.

**Book anchor:** Companion to Supplement A §7.3 (parity, syndrome, Hamming codes).
