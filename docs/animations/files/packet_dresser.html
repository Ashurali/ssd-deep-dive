The packet dresser & ACK/NAK lab

A TLP gets dressed by three layers, undressed by three layers — and between them sits a hostile wire. Includes Ch7's Jammer as a resident gremlin.

Transaction layer adds Header + ECRC, Data Link adds Seq# + LCRC, Physical adds framing. The receiver undresses in reverse — each layer checks before it strips. (Figs 5-11→5-14, 5-19/5-20)

Why does a mere forwarder implement all three layers? Because to route, the switch must undress the packet all the way to the Transaction layer to read the address — then dress it again for the next hop. (Figs 5-15/5-16)

Jammer 👹:

The sender keeps every unacknowledged TLP in its Replay Buffer. Corrupt → NAK → replay. Dropped → receiver sees the sequence jump → NAK → replay. Delayed ACK → sender times out and re-sends → receiver sees a lower-than-expected seq → silently discards the duplicate and ACKs. Three faults = the whole §5.8 state machine. (Figs 5-49→5-53; Jammer = Figs 7-15/7-16)

The receiver advertises buffer credits via FC DLLPs; the sender's gate closes when credits hit zero. That's why a sender can't "just blast". (Figs 5-54/5-55)
