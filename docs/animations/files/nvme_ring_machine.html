The NVMe ring machine

Submission queue, completion queue, doorbells — the whole 8-step dance, with the two details everyone gets wrong: the phase-tag flip and the piggybacked SQ head.

completed: 0

Submit puts a command in the SQ (host memory) and advances the tail; the tail value is then written to the SSD's doorbell register — that write is the doorbell. Watch CQ entries carry a colored phase bit: the host finds new completions by color, and the expected color flips every ring wrap (Fig 6-20). Each completion also carries the current SQ head home (Fig 6-19) so the host knows how far the SSD has consumed.
