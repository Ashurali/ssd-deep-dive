Enumeration & routing explorer

How software discovers a PCIe tree (the all-1s BAR trick), and how a TLP finds its way through it.

boot: nothing mapped yet

Enumeration: for each BAR, software reads, writes all-1s, reads back — the bits that stayed zero are hardwired, and they spell out the region size. Then it allocates host addresses and writes the base in. Routing: every switch port owns a [Base, Limit] address window (shaded); a TLP descends the tree only where its address fits — aim outside every window and it's rejected as an Unsupported Request.
