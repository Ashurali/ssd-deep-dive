# Animation gallery

Self-contained interactive visualizations, each realizing figures from the
book as living diagrams. All run embedded; every wrapper page also has a
full-screen link. Ten more are planned per the
[figure atlas roadmap](../atlas/figure-atlas-animation-roadmap.md).

    `ecc` · `bch` · `ldpc` · `soft-decision`
